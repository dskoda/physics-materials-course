# Physics of Materials - course contents

Course website for UCLA's MAT SCI 120 (Winter 2026), built with Jekyll using the [Cayman theme](https://github.com/pages-themes/cayman).

**Instructor:** Daniel Schwalbe-Koda

## Contents

18 lectures covering electrical, optical, and magnetic properties of solids: free electron model, band theory, semiconductors, devices, dielectrics, and magnetism. Each lecture includes slides, readings, interactive demos, and videos.

## Repositories

- [dskoda/physics-materials-course](https://github.com/dskoda/physics-materials-course) - lecture slides and course materials
- [dskoda/physics-materials-demos](https://github.com/dskoda/physics-materials-demos) - interactive visualizations
- [dskoda/electron-slingshot](https://github.com/dskoda/electron-slingshot) - electrostatics game

## Local preview

```sh
bundle exec jekyll serve
```
